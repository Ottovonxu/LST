# LST
PyTorch Implementation for Locality Sensitive Teaching


Required Package:

    pip install torch==1.10
    pip install scipy 
    pip install cupy-cuda102 
    pip install pynvrtc 
    pip install Cython 

Before Running: Open terminal at this folder

    cd lsh-imt
    make
    cd ..

For running:

    python lsh_space.py
